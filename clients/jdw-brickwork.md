## Build Notes (Josh's Mark Variant)

Different model from scaffolding/flooring/bat-surveys Marks — Josh sells brickwork packages into main contractors on large developments, not directly to site owners.

**What Mark will do:**
- Find new-build developments (residential + commercial) where brickwork packages are coming up
- Target buyers and QSs at the main contractors and developers (not the site owners)
- Reach them after planning approval but before bricklayer appointment — timing window matters
- Cover the full range from 5-unit schemes up to PLCs

**Spec considerations (Andy flagged this as "a different beast"):**
- Lead sources will include planning portals, NHBC, and construction press — not just PlanIt
- Need logic for identifying the main contractor on each development
- Need buyer/QS contact-finding (separate from the owner-operator contact pattern used in Hawks/Bats)
- Spec doc to be written up for Monday 20 April kickoff
