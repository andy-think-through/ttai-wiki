# Wiki Inbox

Drop quick notes here for the daily wiki ingest to pick up. Any `.md` or `.txt` file in this folder will be processed and moved to the appropriate `raw/` subfolder.

Format doesn't matter — bullet points, full paragraphs, stream of consciousness. Just include enough context that future-you knows what domain it relates to.

Examples:
- "Mark-Lite Northants: brickwork firms respond 2x to Wednesday sends vs Monday"
- "AutoStrategy: removed streak dampening from NBA model, +0.7% ROI improvement"
- "Consulting: AccXel discovery day went well, they want enrolment workflow automation"
